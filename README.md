# GitHub Badge Generator

Generate custom GitHub README badges instantly. No server required — runs entirely in the browser.

**[→ Open the app](https://YOUR_USERNAME.github.io/GitIcoGenerator/)** (replace with your GitHub username after you enable Pages)

## Features

- Custom label and status text
- Label and status colors (picker + hex)
- Optional icons: Windows, Linux, macOS, GitHub, Docker, Node
- Live preview
- Copy raw SVG, Markdown image, or img tag
- Download SVG
- Share URL with query params
- Dark/light theme

## Local development

Open `index.html` in a browser, or serve the folder with any static server:

```bash
npx serve .
# or: python -m http.server 8000
```
